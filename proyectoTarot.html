Diario de Tarot

Aplicación web de ocio

Estado: En desarrollo

Aplicación personal para registrar lecturas de tarot y reflexiones en un diario digital.
Este proyecto lo inicié como un regalo para una amiga, con el objetivo de crear una herramienta sencilla y visual, que en el futuro, pueda evolucionar hacia un sistema con almacenamiento persistente y versión móvil.

¿Qué he aprendido?

  • Repaso de HTML y CSS
  • Ha sido el primer proyecto en el que he usado JavaScript
  • Manipulación del DOM
  • Primer proyecto en el que he usado Git
